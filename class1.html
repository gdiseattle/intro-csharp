Variables

What are variables?

C# is a statically typed language with type inference

You must declare all variables before you use them. The declaration must include the variable type and the variable name.

Primitive Variables

A primitive variable is the most basic type of value in a language.

Primitive types we'll be looking at include:

  • int
  • double
  • bool

An expression is a piece of code that evaluates to a value.

Example:

Switch Statement

A switch statement evaluates the value of an integer or string and executes all the statements following the matching "case" block.

In other words, it evaluates the conditional once and starts execution from there until it is told to stop or hits the end of the block

switch (deliveryTimeInHours) {
	case 1:
		Console.Write("Your pizza will arrive in 1 hour");
		break;
	case 2:
		Console.Write("We're busy! Your pizza will arrive 2 hours");
		break;
	default:
		Console.Write("Delivery time unknown");
		break;
}

Without any break statements, the switch blocks "falls through" and executes everything after the matching value too!

Loops

With loops, we can write code once and have it executed multiple times in a row.

For Loop

In cases where we know how many times we wish to execute the loop, we can use a for loop

The for loop structure calls for a variable initialized to a value, the termination condition, and the increment


for (initial variable; test condition; increment) { }

Develop it!


for (int orderCost = 0; orderCost < 50; orderCost +=10) {
	Console.Write("My order costs $" + orderCost + "!");
}

Output


My order costs $0!
My order costs $10!
My order costs $20!
My order costs $30!
My order costs $40!

While Loop

At other times, we want to execute our code for an unknown number of iterations. We want to execute it while an expression is true regardless of how many iterations it takes.


while (condition) {
...
}

Develop It!

While isOrderFull is true, add a pizza, then update the value of isOrderFull

The Code


isOrderFull = numberOfPizzas == maxNumberOfPizzas;
Console.Write("Our max number of pizzas is " + maxNumberOfPizzas);
while (!isOrderFull) {
	/* We'll add a pizza */
	numberOfPizzas ++;
	/* We'll alert how many pizzas we have */
	Console.Write("I'm carrying: " + numberOfPizzas);
	/* We'll update whether or not we're full */
	isOrderFull = numberOfPizzas == maxNumberOfPizzas; 
}
